116/117. Populating Next Right Pointers in Each Node

Posted on 2023-01-06 | In Leetcode

You are given a perfect binary tree where all leaves are on the same level, and every parent has two children. The binary tree has the following definition:

struct Node {
int val;
Node *left;
Node *right;
Node *next;
}
Populate each next pointer to point to its next right node. If there is no next right node, the next pointer should be set to NULL.

Initially, all next pointers are set to NULL.

Use stack to implement BFS, spereate each level with None on stack, for each node poped from stack, point their next to be the first item on stack.

115. Distinct Subsequences

Posted on 2023-01-06 | In Leetcode

Given two strings s and t, return the number of distinct subsequences of s which equals t.

The test cases are generated so that the answer fits on a 32-bit signed integer.

First thought is to use backtracking, and add 1 to answer when a valid path is found, but took to much time and failed text cases with time limit exceeded. Code below

Use Dynamic Programming, where dp[i][j] stores number of distinct subsequences for s[i:] t[j:], it is noticed that if s[i] != t[j], dp[i][j] = dp[i+1][j].
if s[i] = t[j], dp[i][j] = dp[i+1][j]+dp[i+1][j+1] where dp[i+1][j] are subsets that do not use s[i] to match t[j], and dp[i+1][j+1] is to use s[i] in the subset.
It is observered we only need dp roe i+1 whe computing row i, so we only need one 1 d dp table.

112. Path Sum

111. Minimum Depth of Binary Tree

The minimum depth is the number of nodes along the shortest path from the root node down to the nearest leaf node.

Note: A leaf is a node with no children.

Use recursion to find min tree height, when there is no child nodes, return one, else return 1 + minimum height of left and right child.

110. Balanced Binary Tree

Posted on 2023-01-05 | In Leetcode

Given a binary tree, determine if it is height-balanced.

Use recursion to find tree height, for each node, by adding 1 to the max depth of it’s subtrees.
Compare height of each node to see if difference is greater than 1.

For a height balanced binary search tree, the root is the middle number of values of it’s subtrees, so we choose the value in the middle of nums to be the root, and compute left and right subtree from left and right half of nums.
